tests.test_gaussian_imputer.dummy_model(x)[source]¶

A dummy model that returns the sum of the features.

Note

This callable is just here that we satisfy the Imputer’s model parameter and tha we can check if the Imputer can be called with coalitions and returns a vector of “predictions”.

Parameters:

x (ndarray) – Input data as a 2D numpy array with shape (n_samples, n_features).

Returns:

A 1D numpy array with the sum of the features for each sample.

ndarray
